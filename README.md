# CSS Grid Playground 🎨

A production-ready, interactive live editor for CSS Grid. Experiment with grid layouts in real-time, use preset templates, and learn common properties with a built-in cheatsheet.

![CSS Grid Playground Placeholder](https://via.placeholder.com/1200x600/121212/bb86fc?text=CSS+Grid+Playground+Live+Editor)

## ✨ Features

- **Live Real-time Editing:** See your changes instantly as you type CSS in the editor.
- **Dual Editor Panes:** Separate control for the Grid Container and individual Grid Items.
- **Dynamic Item Management:** Add or remove grid items with the click of a button (+ / -).
- **Preset Templates:** Start with common layouts:
  - Basic 3-Column
  - Holy Grail Layout
  - Mosaic Layout
  - 12-Column System
  - Responsive Auto-fill
- **Interactive Cheatsheet:** Quick reference for CSS Grid properties with one-click copy.
- **Error Handling:** Real-time syntax validation with friendly error messages.
- **Dark Theme UI:** Modern, clean, and developer-friendly aesthetic.
- **Export Ready:** One-click "Copy CSS" button to take your generated code to your own project.

## 🚀 How to Run Locally

This project is built with vanilla HTML, CSS, and JavaScript. No installation or build steps are required.

1. Clone this repository:
   ```bash
   git clone https://github.com/elesiaann/css-grid-playground.git
   ```
2. Navigate to the project folder:
   ```bash
   cd css-grid-playground
   ```
3. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Modern properties, Flexbox for app layout, and CSS Grid for the playground.
- **JavaScript (ES6+):** Pure vanilla JS for the live engine and state management.
- **No Dependencies:** Zero external libraries or frameworks.
